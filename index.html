CSS BOX MODEL ASSIGNMENT

1.A box model is a box that wraps around every Html element. It consists of margins,borders,padding and actual content.

    2.PROPERTIES OF A BOX MODEL

  1. content (consists of texts and images)
  2. Padding.(Its the space between the content)
  3. Border.(This can be coloured and styled)
  4. Margin.(This is the space outside of the box)

    3.calculating the total width of a paragraph given the following
  1. width=400px
  2. padding=20px
  3. margin=20px
  4. Border=2px
